# 国际化功能演示

## 🎬 演示场景

### 场景 1：首次访问（浏览器语言检测）

**中文浏览器用户**
```
1. 用户首次访问应用（无 Cookie）
2. 浏览器语言设置为中文（zh-CN）
3. 应用自动检测并显示中文界面
4. 登录页显示："智能体平台"、"登录您的账户"
```

**英文浏览器用户**
```
1. 用户首次访问应用（无 Cookie）
2. 浏览器语言设置为英文（en-US）
3. 应用自动检测并显示英文界面
4. 登录页显示："Agent Platform"、"Sign in to your account"
```

### 场景 2：手动切换语言

**在登录页切换**
```
1. 用户访问登录页
2. 点击右上角的语言切换按钮（地球图标 🌐）
3. 在下拉菜单中选择"中文"或"English"
4. 页面立即切换到所选语言
5. Cookie 自动保存语言选择
```

**在主界面切换**
```
1. 用户登录后进入主界面
2. 在 Header 右侧找到语言切换按钮（用户菜单左侧）
3. 点击并选择语言
4. 整个应用界面立即切换
5. 导航到其他页面，语言保持一致
```

### 场景 3：语言持久化

**刷新页面**
```
1. 用户选择中文
2. 刷新浏览器页面
3. 页面仍然显示中文（从 Cookie 读取）
```

**关闭并重新打开浏览器**
```
1. 用户选择中文
2. 关闭浏览器
3. 第二天重新打开浏览器访问应用
4. 页面仍然显示中文（Cookie 有效期 365 天）
```

## 📊 数据流

### Cookie 数据流

```
用户选择语言
    │
    ▼
┌─────────────────────┐
│ i18n.changeLanguage │
│      (语言代码)      │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ Cookies.set()       │
│ 'app_language', lng │
│ expires: 365 days   │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 浏览器 Cookie 存储  │
│ app_language=zh     │
└─────────────────────┘
```

### 页面刷新数据流

```
页面加载
    │
    ▼
┌─────────────────────┐

└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 语言检测器          │
│ detect()            │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ Cookies.get()       │
│ 'app_language'      │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 返回语言代码        │
│ 'zh' 或 'en'        │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 加载对应翻译文件    │

└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 渲染页面            │
└─────────────────────┘
```

## 🎨 视觉效果

### 语言切换动画
- 点击按钮：下拉菜单淡入（100ms）
- 选择语言：文本即时更新（无动画）
- 菜单关闭：淡出（75ms）

### 当前语言指示
- 下拉菜单中当前语言有蓝色背景
- 右侧显示勾选标记 ✓

### 响应式设计
- 桌面：显示完整语言名称（"English" / "中文"）
- 移动：仅显示图标（🌐）

## 💡 用户体验亮点

1. **零学习成本** - 直观的地球图标，国际通用
2. **即时反馈** - 语言切换立即生效，无需等待
3. **智能默认** - 自动匹配用户浏览器语言
4. **持久记忆** - 一次选择，长期有效
5. **全局一致** - 所有页面保持相同语言
